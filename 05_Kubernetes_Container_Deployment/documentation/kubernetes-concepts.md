# Kubernetes Concepts

## Namespace
The namespace isolates Kubernetes resources for this project:
- cloud-portfolio

## Deployment
Deployments manage:
- pod replicas
- updates
- self-healing containers

## Pods
Pods run the Flask application containers.

## Services
The LoadBalancer service exposes the application externally.

## Containerization
Docker packages:
- application code
- dependencies
- runtime environment

## Scaling
Kubernetes supports horizontal scaling through replica configuration.
