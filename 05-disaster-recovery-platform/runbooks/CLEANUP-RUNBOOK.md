# Cleanup Runbook - Disaster Recovery Platform

## AWS
Delete EC2 instances, NAT gateways, Elastic IPs, route tables, VPC flow logs, CloudWatch alarms, backup plans, and VPC resources.

## Azure
Delete the project resource group after confirming evidence screenshots have been saved.

## GCP
Delete the Compute Engine instances, firewall rules, VPC network, disks, and static IPs. For full cleanup, shut down the GCP project.
