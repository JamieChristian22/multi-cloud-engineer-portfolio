# Architecture - Monitoring Platform

## Goal

Create a multi-cloud observability platform that proves the ability to monitor cloud infrastructure, collect logs, configure alarms, and route alerts.

## AWS
- CloudWatch metrics and alarms
- SNS topic for notifications
- VPC Flow Logs
- EC2 CPU alarm

## Azure
- Azure Monitor metrics
- Log Analytics workspace
- Action Group email notification
- VM insights or metrics alert

## GCP
- Cloud Monitoring alert policy
- Uptime check
- Log-based metric
- Notification channel
