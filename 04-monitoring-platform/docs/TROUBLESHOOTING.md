# Troubleshooting - Monitoring Platform

## Common Issues

### Public web page does not load
Check VM state, external IP, firewall/security group rules, network tags, and web server status.

### SSH fails
Check key permissions, username, public IP, route table, NSG/security group/firewall rule, and VM agent status.

### Cloud resource not visible
Confirm the correct account, subscription, tenant, project, region, and directory are selected.

### Cost warning appears
Stop or delete test VMs, NAT gateways, bastion hosts, static IPs, and unattached disks when not in use.
