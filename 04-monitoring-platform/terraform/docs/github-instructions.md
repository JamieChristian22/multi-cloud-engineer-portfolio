# How to Add This to GitHub

1. Unzip this package.
2. Copy the `terraform` folder into:

```text
multi-cloud-engineer-portfolio/04-monitoring-platform/
```

3. Your final path should be:

```text
multi-cloud-engineer-portfolio/04-monitoring-platform/terraform/
```

4. From the repo root, run:

```bash
git status
git add 04-monitoring-platform/terraform
git commit -m "Add Terraform for monitoring platform"
git push
```

5. Add screenshots to your evidence folder.
